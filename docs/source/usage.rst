.. _install:

Install
-------------

To use avt, first install it using pip:

.. code-block:: console

   (.venv) $ pip install git+https://github.com/alexcapstick/alexs-visualisation-tools@main#egg=avt



For example::

   >>> import avt


From here, you can use the functions and classes provided, for example::

   >>> avt.clockplot()
